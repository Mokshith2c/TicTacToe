# 🎮 Tic-Tac-Toe Game (Emoji Edition 🐯🦁)

A fun and interactive Tic-Tac-Toe game built using **HTML, CSS, and JavaScript**, where players compete using emojis instead of traditional X and O.

---

## 🚀 Features

* 🐯 vs 🦁 Emoji-based gameplay
* 👥 Two-player mode (same device)
* 🎯 Automatic winner detection
* 🤝 Draw detection with message
* 🔄 Reset and New Game functionality
* ⚡ Simple and fast UI

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla JS)

---

## 🎯 How to Play

1. Player 1 uses **🐯**
2. Player 2 uses **🦁**
3. Click on any empty box to make a move
4. First player to align 3 in a row wins:

   * Horizontal
   * Vertical
   * Diagonal
5. If all boxes are filled and no winner → It's a **Draw**

---

## ⚙️ How It Works

* Tracks turns using a boolean (`turnO`)
* Stores all winning combinations in `winPatterns`
* Checks winner after every move
* Highlights winning boxes
* Disables all boxes after win/draw
